# BlogCreate 사용법

## 1. git clone

git clone

## 2. npm i

cd client, cd server 에서 각각 npm i

## 3. server/config/config.js 수정

mysql username부분 자신의 mysql username으로 수정

## 4. server/.env파일 생성

COOKIE_SECRET=쿠키명  
MYSQL_PASSWORD=mysql비밀번호

## 5. server npm run dev

cd server, npm run dev  
cd client, npm run dev
